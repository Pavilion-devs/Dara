Anoncoin Third Party APIs
Overview
These are the Anoncoin Third Party APIs

Read documentation carefully before you start api integration.

Let us know if you have any suggestion to improve this documentation.

API Host
Dev Host: https://dev-api.dubdub.tv
Prod Host: https://api.dubdub.tv

SOLANA RPC:
DEV RPC URL: https://api.devnet.solana.com
PROD RPC URL: https://api.mainnet-beta.solana.com

Authentication
All authorized requests should be sent with the following headers:

Authorization: API-KEY

How to get the API Key
Log in to your account at https://anoncoin.it

Click on your profile avatar in the top-right

Go to API Key

Click Generate API Key

POST
Create Token
{{HOST}}/thirdParty/v1/createToken
Creates a new token on the Solana blockchain.

Success - Create Token
javascript
var myHeaders = new Headers();
myHeaders.append("x-api-key", "anoncoin:xxxx");

var formdata = new FormData();
formdata.append("files", fileInput.files[0], "1eec0f62-7619-4b20-abf3-de18b41eb4eb");
formdata.append("tickerName", "Thanos");
formdata.append("tickerSymbol", "BOLD51");
formdata.append("description", "Destroyed half of the world");
formdata.append("twitterLink", "https://x.com/abc");
formdata.append("telegramLink", "https://t.me/abc");
formdata.append("royaltyUser", "anoncoinit");

var requestOptions = {
  method: 'POST',
  headers: myHeaders,
  body: formdata,
  redirect: 'follow'
};

fetch("{{HOST}}/thirdParty/v1/createToken", requestOptions)
  .then(response => response.text())
  .then(result => console.log(result))
  .catch(error => console.log('error', error));

Response
Success Response (when validateOnly is false or not provided)
json
{
    "mintAddress": "token mint address",
    "signedTransaction": "signed transaction in base 58 encoded string",
    "blockhash": "blockhash in signed transaction",
    "lastValidBlockHeight": "blockhash valid till block height",
    "blockhashExpiresInSeconds": "blockhash expiry in seconds [number]"
}
Response Fields
Field	Description
mintAddress	The Solana token mint address for the new token
signedTransaction	The signed transaction in base58 encoded string format, ready to be broadcast to the Solana network
blockhash	The blockhash used for the transaction
lastValidBlockHeight	The block height until which the transaction remains valid
blockhashExpiresInSeconds	Time in seconds until the blockhash expires (typically 50-60 seconds)
